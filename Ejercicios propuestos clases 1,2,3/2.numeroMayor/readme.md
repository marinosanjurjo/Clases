
# ¿Qué número es mayor?

Introducir con dos prompts dso números.
Número 1 y número 2

Si el número 1 es MAYOR que el número 2 escribir en un DIV del HTML

```
El número 1 (6) es MAYOR que el número 2 (3)
```

Si el número 1 es MENOR que el número 2 escribir en un DIV del HTML

```
El número 1 (6) es MENOR que el número 2 (10)
```

Si el número 1 es igual al número 2

```
El número 1 (6) es igual al número 2 (6) 
```

## Mejoras

Sustituir los prompts por dos inputs y añadir un botón de Calcular
