# Número impar

Introducir un input de tipo number con un botón

Cunado se clicke en el botón calcular si el número es PAR o IMPAR y escribirlo en un contenedor de la web.

Si el número es PAR
````
El 4 es PAR
````

````
El 3 es IMPAR
````

Si es cero

````
¡¡ES CERO!
````

Si no hay dato en el input

```
No hay número introducido
```

## Mejoras

1. Añadir clases
    
    Añadir una clase 'verde' al contenedor cuando es PAR

    Añadir una clase 'azul' cuando el valor es IMPAR

    Añadir una clase 'amarillo' cuando es CERO

    Añadir un clase 'rojo' cuando no hay número introducido

2. Añadir otro contenedor para decir además si es múltiprlo de 3

````
El número 12 es PAR y múltiplo de 3

El número 11 es impar y no es múltiplo de 3

````
