# Pop up

Maquetar un pop up 

## Recomendaciones 

1. Maquetar el pop up abierto. Recordad que un pop up tiene que estar maquetado por "encima" de todo lo demás. Fijo en la pantalla

2. El pop up que ocupe toda la pantalla, y que tenga fondo semi-transparente (un color con tranparencia). Dentro del pop up maquetar la zona interior del pop up (en la imagen lo que tiene el fondo blanco).

3. Una vez hecho pensar cómo gestionar el JS para añadir o no, la cualidad que le hace visible
