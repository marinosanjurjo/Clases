# Suma de los números impares que hay de 0 a 100

El programa debe calcular la suma de todos los números impares que hay de 0 a 100.

Pensar el programa suponiendo que podamos cambiar de 100 a cualquier número.

## Mejora 1
Que se pulse a un botón se calcule y se pinte el resultado en la web

## Mejora 2
Incluir dos inputs, para añadir el número mínimo y el número máximo.

Adaptar el programa para que el usuario pueda calcular la suma de los números impares entre dos números introducidos.
