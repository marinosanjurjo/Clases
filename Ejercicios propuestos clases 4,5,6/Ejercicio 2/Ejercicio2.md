# Cálculo de diámetro, perímetro y área de un círculo mediante funciones

Crea una interfaz con HTML y CSS para introducir el radio del círculo.

Con un botón se debe calcular el diámetro, el perímetro y el área mediante ***funciones separadas***

````
# Fórmula de diámetro

2 * radio

# Fórmula del perímetro

2 * pi * radio

# Fórmula del área

pi * radio * radio

````
